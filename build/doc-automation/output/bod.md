# 1. Übersicht
Die Geschäftsleitung ist die oberste Entscheidungsinstanz für strategische und strukturelle Datenkonflikte. Die Geschäftsleitung entscheidet entlang der Organisationsstrategie – nicht entlang bestehender Strukturen.

- Entscheidung bei Grundsatzfragen der Ownership
- Löst strategische Zielkonflikte zwischen Bereichen auf.
- Schirmherrschaft bei politischen / reputationskritischen Themen.

👉 Entscheidet strategisch / selten / richtungsweisend

# 2. Typische Einsatzfälle
- Wer ist 🧑‍✈️🧭 Dateneigner bei zentralen Datendomains?
- Einführung einer "Single Source of Truth"
- Auflösung historischer Silos
- Konflikte mit politischer Implikation

# 3. Entscheidungsprinzip
👉 Die Geschäftsleitung entscheidet entlang der Organisationsstrategie – nicht entlang bestehender Strukturen.

# 4. RACI - Geschäftsleitung bei "Fach vs. Fach"

|---|---|---|---|---|---|---|
| Problem identifizieren | C | R | I | C | C | **I** |
| Analyse &amp; Optionen erarbeiten | C | R | C | R | C | **I** |

| Entscheidungen vorbereiten | C | C | I | R | R | **C** |

| Umsetzung mandatieren |  R | R | R |R | C | **A** |


# 5. Verantwortung
- Klärt Ownership-Fragen endgültig
- Setzt organisationale Prioritäten
- Trifft Entscheidungen bei
  - Zielkonflikten
  - Ressourcenverteilung
  - politischer Relevanz
