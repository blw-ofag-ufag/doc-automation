# Module RACI Matrix

# Zielpublikum
- Fachorganisation - 👩‍💼💡 Datenverwalter (Data Steward)
- IT-nahe Organisationen - 🕵🏻⚙️ Datenhalter (Data Custodian)

---

# Einsatzscope
Die Datengouvernanz stellt sicher, dass Daten als strategische Ressource gesteuert, verantwortet und genutzt können.. Die erweiterten Module werden insbesondere benötigt bei
- IT-Betrieb und Planung
- komplexeren IT- oder Datenprojekte

---

# 1. RACI-Matrix - erweitertes Modell (mit Modulen)

## Legende
- R (Responsible) = führt aus
- A (Accountable) = trägt Gesamtverantwortung
- C (Consulted) = wird einbezogen
- I (Informed) = wird informiert

## 1.1 Governance &amp; Strategie

| Aktivität | 🧑‍✈️🧭 Dateneigner | 👩‍💼💡 Datenverwalter | 🕵🏻⚙️ Datenhalter | Gremium Datengouvernanz (GDG) 👥🧭 |
|---|---|---|---|---|
| Datenstrategie definieren | A | I | I | R |
| Data Governance Policies verabschieden | A | I | I | R |
| Priorisierung Dateninitiativen | A | R | I | C |
| Kritische Daten definieren |  A | R | I | C |


## 1.2 Datennutzung &amp; Zugriff
| Aktivität | 🧑‍✈️🧭 Dateneigner | 👩‍💼💡 Datenverwalter | 🕵🏻⚙️ Datenhalter | Datenschutzberater 👤⚖️
|---|---|---|---|---|
| Zugriff auf Daten genehmigen | A | C | R | C |
| OGD-Freigabe | A | R | I | C |
| Datennutzung definieren | A | R | I | C |

## 1.3 Architektur &amp; IT-Betrieb
| Aktivität | 🧑‍✈️🧭 Dateneigner | 👩‍💼💡 Datenverwalter | 🕵🏻⚙️ Datenhalter | Anwendungsverantwortliche 👤⚙️ | Datenarchitekt 👤📐 |
|---|---|---|---|---|---|
| Datenmodell definieren | I | C | C | R | A/R |
| Datenintegration umsetzen | I | C | R | R | A |
| Betrieb sicherstellen | I | I | A/R | C | I |
| Backup &amp; Recovery | I | I | A/R | C | I |

## 1.4 Compliance &amp; Sicherheit
| Aktivität | 🧑‍✈️🧭 Dateneigner | 🕵🏻⚙️ Datenhalter | ISBO (Informationssicherheitsbeauftragter der Organisation) 👤⚠️ | Datenschutzberater 👤⚖️ |
|---|---|---|---|---|
| Schutzbedarf festlegen | A | C | R | C |
| Sicherheitsmassnahmen umsetzen | I | R | A | C |
| Datenschutz-Compliance prüfen |  A | C | C | R |
| Incident Management | I | R | A | C |
