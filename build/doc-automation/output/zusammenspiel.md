# Einfaches Zielbild
- 🧑‍✈️🧭 Dateneigner (Data Owner) entscheidet WARUM und WOFÜR
- 👩‍💼💡 Datenverwalter (Data Steward) definiert WAS und WIE fachlich
- 🕵🏻⚙️ Datenhalter (Data Custodian) setzt WAS und WIE technisch um

---

# Beispiel: Datenqualitätsproblem
1. Data User 👤⁉️ meldet ein Problem.
2. 👩‍💼💡 Datenverwalter (Data Steward) analysiert die Ursache.
3. 🕵🏻⚙️ Datenhalter (Data Custodian) prüft technische Aspekte.
4. 🧑‍✈️🧭 Dateneigner (Data Owner) entscheidet bei grösserem Impact.

# Zentrale Regeln
👉 Keine Rolle funktioniert isoliert – Datengouvernanz ist Zusammenarbeit.
👉 Aufgaben und Kompetenzen können delegiert werden, nicht aber Verantwortung.
